# TrendPullbackEA — MQL5 Expert Advisor

Консервативный трендовый советник «pullback по тренду». Не обещает иксов за месяц — обещает понятную математику и фиксированный риск.

## Логика стратегии

Сделка открывается, когда **одновременно** выполняются 4 условия:

1. **Тренд есть.** Цена выше EMA(200) для лонга / ниже для шорта.
2. **Тренд сильный.** ADX(14) > 20 (порог настраивается).
3. **Откат к EMA(50).** Расстояние от цены до EMA(50) ≤ 1.5 × ATR(14) — то есть мы покупаем не на хае, а на откате.
4. **Импульс развернулся.** RSI(14) пересекает 40 снизу вверх (лонг) или 60 сверху вниз (шорт) на последнем закрытом баре.

**Выход:**
- Stop-loss = 1.5 × ATR(14) от цены входа.
- Take-profit = 3.0 × ATR(14) (соотношение риск/прибыль = 1:2).
- Трейлинг включается после прохода 1R прибыли, дистанция трейла = 1.5 × ATR.

**Защита капитала:**
- Риск на сделку — 1% от equity (настраивается).
- Дневной лимит убытка — 4% (после превышения — никаких новых сделок до конца дня).
- Максимум 1 открытая позиция на инструмент.
- Размер лота рассчитывается от дистанции стопа и tick-value инструмента, а не как «фиксированные 0.01».

## Где использовать

| Параметр | Рекомендация |
|---|---|
| Таймфрейм | H1 (можно H4) |
| Инструменты | XAUUSD, US30, NAS100, EURUSD, GBPUSD, USDJPY |
| Минимальный депозит | $1000+ для full lot; $100 — только на cent-аккаунте |
| Брокер | ECN с низким спредом; разрешён hedging *или* netting |

⚠️ На депозите $100 классический счёт **не подойдёт**: минимальный лот 0.01 + спред съедают весь риск-бюджет. Используй **cent-account** (1 цент = 1 доллар внутреннего расчёта) — там $100 эквивалентны $10 000 по логике лотов.

## Установка

1. Скопируй `TrendPullbackEA.mq5` в `<папка_MT5>/MQL5/Experts/`.
2. Открой MetaEditor → F7 (Compile). Ошибок быть не должно.
3. В терминале MT5: перетащи советник на график инструмента нужного таймфрейма.
4. Во вкладке «Common» включи «Allow Algo Trading». Во вкладке «Inputs» проверь параметры.

## Бэктест перед реальной торговлей

**Обязательно** прогони перед заливом денег:

1. Strategy Tester → Model: **Every tick based on real ticks**.
2. Период: минимум 5 лет (лучше 10).
3. Спред: **Current** или фиксированный реалистичный для твоего брокера.
4. Что смотреть в отчёте:
   - **Profit Factor** ≥ 1.3
   - **Sharpe** ≥ 1.0
   - **Max Drawdown** ≤ 20–25%
   - **Recovery Factor** ≥ 2
   - **Trades** ≥ 100 (иначе статистика недостоверна)
5. **Walk-forward.** Оптимизируй на первой половине истории, проверь на второй половине без переподгонки. Если результат рассыпается — стратегия переоптимизирована, не торгуй.

## Параметры (основные)

| Input | По умолчанию | Что делает |
|---|---|---|
| `InpRiskPercent` | 1.0 | Риск на сделку, % от equity |
| `InpMaxDailyLossPct` | 4.0 | Стоп торговли после потери X% за день |
| `InpEmaTrendPeriod` | 200 | EMA-фильтр тренда |
| `InpEmaPullbackPeriod` | 50 | EMA, к которой ждём откат |
| `InpAdxThreshold` | 20.0 | Минимальный ADX для входа |
| `InpRsiLongLevel` / `InpRsiShortLevel` | 40 / 60 | RSI-уровни для подтверждения |
| `InpAtrStopMult` | 1.5 | Стоп = N × ATR |
| `InpAtrTargetMult` | 3.0 | Тейк = N × ATR |
| `InpUseTrailing` | true | Включить трейлинг после 1R |

## Реалистичные ожидания

Это **не** «300% в месяц». Цели для такой стратегии:
- 30–80% годовых на хорошей дистанции при просадке 15–25%.
- Win-rate 40–50%, средняя сделка ≈ 1.5R (за счёт RR 1:2 и трейлинга).
- Серии из 5–7 убыточных сделок подряд — нормально, это математика.

Если за 2 месяца торговли видишь -10% — это **в пределах ожидаемого**, не выключай и не «улучшай» советник на эмоциях.

## Чего советник НЕ делает

- ❌ Не использует мартингейл, сетки, усреднения.
- ❌ Не торгует без стоп-лосса.
- ❌ Не «отыгрывает» убытки увеличением лота.
- ❌ Не торгует по новостям и сразу после открытия рынка (входит только на пересечении RSI, что естественно фильтрует первые волатильные тики).

Если хочешь добавить новостной фильтр, sessions filter, или мульти-таймфрейм подтверждение — это следующий шаг после того, как протестируешь базовую версию.
